# Code of Conduct

This project adopts the
[**Contributor Covenant, version 2.1**](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. The full text is canonical and lives at the URL
above; please read it before participating.

In short: be respectful, assume good faith, and help make this a welcoming
project for people of every background and level of experience.

## Reporting

If you experience or witness behaviour that violates the Covenant, contact
the maintainer at **[david@oakstack.eu](mailto:david@oakstack.eu)**.

Reports are handled confidentially. The maintainer will acknowledge within
a few business days, investigate, and decide on a response — which may
range from a private conversation to a temporary or permanent ban from the
project's spaces, in line with the Covenant's enforcement guidelines.

## Scope

This Code of Conduct applies in all project spaces — the GitHub
repository (issues, pull requests, discussions, code review), the
project's chat channels if/when they exist, and any public event or
medium where someone is representing the project.
